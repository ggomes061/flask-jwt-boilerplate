# Secure REST API Boilerplate with Flask + JWT

Production-ready backend with:
- JWT Authentication (access + refresh tokens)
- Role-based access (admin/user)
- PostgreSQL + SQLAlchemy
- Rate limiting & CORS
- Docker support
- Full Postman collection

Live demo: https://flask-jwt-boilerplate.onrender.com  
Documentation: /docs (Redoc)

Tech stack: Python 3.11 • Flask • PyJWT • PostgreSQL • Docker
